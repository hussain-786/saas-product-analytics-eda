## 🎯 Project Overview

This project performs an in-depth exploratory data analysis on a global SaaS enterprise dataset from Zoho. The analysis covers:

- **User Growth Patterns** - Understanding customer acquisition and expansion trends
- **Pricing Models** - Analyzing different pricing strategies and their impact
- **Product Categories** - Evaluating performance across various product segments
- **Customer Ratings** - Assessing customer satisfaction and feedback patterns
- **Market Trends** - Identifying emerging trends in the SaaS industry

The goal is to extract actionable insights that can drive business decision-making, improve customer retention, and optimize pricing strategies.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed along with the required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter openpyxl
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hussain-786/saas-product-analytics-eda.git
cd saas-product-analytics-eda
```

2. Open the Jupyter Notebook:
```bash
jupyter notebook "SaaS Product Analytics Market Trends and Customer Insights.ipynb"
```

3. Run all cells to generate the analysis and visualizations
